# EcoPest Field — تطبيق إيكوبست الميداني

تطبيق Expo مرافق لنظام EcoPest. الهدف منه فتح مسار الفني بسرعة من الهاتف: شاشة مسح، إدخال رقم المحطة يدويًا عند الحاجة، وإرسال تقارير ميدانية. حسابات المدير والمشرف تفتح بوابة الويب الإدارية المناسبة بجلسة مؤقتة آمنة.

## التشغيل

```bash
npm install
npm run start
npm run web
```

انسخ `.env.example` إلى `.env` واضبط:

```bash
EXPO_PUBLIC_ECOPEST_WEB_BASE_URL=https://your-ecopest-domain.com
```

تسجيل الدخول في الموبايل يستخدم Better Auth عبر نفس باك اند الويب. جلسة المستخدم تحفظ في `expo-secure-store`، وطلبات API المحمية ترسل cookie الجلسة المخزنة.

## الملاحظات الأمنية

- لا توجد أسرار قاعدة بيانات أو Gemini داخل تطبيق الموبايل.
- التطبيق يقرأ profile المستخدم النشط من `/api/mobile/me` بعد تسجيل الدخول.
- المدير والمشرف يفتحون بوابة الويب عبر رابط handoff قصير العمر يستخدم مرة واحدة.
- الوضع الداكن مدعوم من داخل التطبيق: النظام، فاتح، داكن.

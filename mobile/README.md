# EcoPest Field — تطبيق إيكوبست الميداني

تطبيق Expo مرافق لنظام EcoPest. الهدف منه فتح مسار الموبايل من الهاتف مباشرة: شاشة مسح، إدخال رقم المحطة يدويًا عند الحاجة، إرسال تقارير ميدانية، ومراجعة تقارير المدير والمشرف داخل التطبيق. الموبايل لا يتصل بقاعدة البيانات مباشرة؛ كل البيانات تمر عبر API الويب وتكتب في نفس قاعدة بيانات EcoPest.

## التشغيل

```bash
npm install
npm run start
npm run web
```

خادم الموبايل ثابت على:

```bash
https://ecopest-production.vercel.app/
```

تسجيل الدخول في الموبايل يستخدم Better Auth عبر نفس باك اند الويب. جلسة المستخدم تحفظ في `expo-secure-store`، وطلبات API المحمية ترسل cookie الجلسة المخزنة.

## الملاحظات الأمنية

- لا توجد أسرار قاعدة بيانات أو Gemini داخل تطبيق الموبايل.
- التطبيق يقرأ profile المستخدم النشط من `/api/mobile/me` بعد تسجيل الدخول.
- المدير والمشرف يستخدمون شاشة إدارة Native داخل التطبيق بدل التحويل إلى بوابة الويب.
- الوضع الداكن مدعوم من داخل التطبيق: النظام، فاتح، داكن.
